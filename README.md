# Beyond

日々のトレーニングを記録するアプリです。種目、重量、回数、セット数を記録することで、トレーニング時に過去の記録を参照できるだけではなく、積み上げてきた記録による達成感や自己成長を感じることもできます。

## Features

- ユーザ登録/ログイン認証機能
- CRUD 機能(トレーニング記録、トレーニング種目)
- Storybook, Jest を使ったテスト開発
